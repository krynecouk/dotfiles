## NVIM configuration

### How to install plugins

`nvim +PlugInstall`

### How to autoremove plugins
`nvim +PlugClean`

### How to change theme
`nvim ~/config/nvim/visuals.vim`

### How to install new plugin
`nvim ~/config/nvim/plugins.vim`

### How to configure plugin
`nvim ~/config/nvim/plugin/<plugin>`
